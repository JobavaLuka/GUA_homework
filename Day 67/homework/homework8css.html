CSS-ის შექმნა

CSS (Cascading Style Sheets) შეიქმნა 1996 წელს. მისი მიზანია ვებ-გვერდის ვიზუალური სტილის განსაზღვრა — ფერები, ზომები, განლაგება და შრიფტები.

რატომ გახდა საჭირო CSS?

ადრე დიზაინი მთლიანად HTML-ში იწერებოდა, რაც კოდს რთულსა და არეულს ხდიდა. CSS-მ გამოყო სტილი სტრუქტურისგან, რის შედეგადაც კოდი გახდა სუფთა და ადვილად სამართავი.

CSS2 და CSS3

CSS2-მ დაამატა უკეთესი პოზიციონირება და მედია ტიპები. CSS3-მ შემოიტანა ანიმაციები, transition-ები, shadow-ები, Flexbox და Grid სისტემა.

CSS-ის მნიშვნელობა დღეს

თანამედროვე ვებ-დიზაინი წარმოუდგენელია CSS-ის გარეშე. ის უზრუნველყოფს responsive დიზაინს, მობილურ მხარდაჭერას და თანამედროვე ვიზუალურ ეფექტებს.
