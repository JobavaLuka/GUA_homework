HTML-ის შექმნა

HTML (HyperText Markup Language) შეიქმნა 1991 წელს ტიმ ბერნერს-ლის მიერ. მისი მთავარი მიზანი იყო ინფორმაციის გაზიარება ინტერნეტში. თავდაპირველად HTML ძალიან მარტივი იყო და შეიცავდა მხოლოდ რამდენიმე ტეგს.

პირველი ვერსიები

1993 წელს გამოვიდა HTML 1.0. მოგვიანებით HTML 2.0 და 4.01, რომლებმაც დაამატეს ცხრილები, ფორმები და უფრო დახვეწილი სტრუქტურა. HTML 4 დიდი ხნის განმავლობაში იყო მთავარი სტანდარტი.

HTML5 - თანამედროვე ეტაპი

2014 წელს ოფიციალურად დამტკიცდა HTML5. ამ ვერსიამ დაამატა ვიდეო და აუდიო ელემენტები, სემანტიკური ტეგები (header, footer, article, section) და უკეთესი მხარდაჭერა მობილური მოწყობილობებისთვის.

HTML-ის მნიშვნელობა

HTML ქმნის ვებ-გვერდის სტრუქტურას. ის განსაზღვრავს სათაურებს, პარაგრაფებს, სურათებს, ბმულებს და სხვა ელემენტებს. CSS და JavaScript სწორედ HTML-ზეა დაფუძნებული.
